# 김유정 | Mobile App Developer

안녕하세요.
AI와 모바일을 연결해 사람들의 일상에 실질적인 도움을 주는
앱 개발자 김유정입니다.

Flutter · Android(Kotlin) 기반 모바일 앱 개발 경험이 있으며
OpenAI API를 활용한 AI 기능 기획 및 구현부터
Spring Boot 백엔드 연동까지 풀스택으로 개발하고 있습니다.

## 🛠 기술 스택

**Mobile**
- Flutter / Dart
- Android (Kotlin, Java)
- React Native

**Backend**
- Spring Boot
- Firebase (Firestore, Authentication)

**AI / API**
- OpenAI API (GPT)

**Database**
- MySQL
- Firebase Firestore

**Tools**
- Git / GitHub
- Figma

## 📱 주요 프로젝트

### 마음이 (Maeum) 🚧 개발중
Flutter 기반 AI 감정 케어 애플리케이션 — 출시 목표

- 현대인의 정신건강 문제를 해결하기 위한 일상 감정 기록 및 AI 분석 서비스
- OpenAI API를 활용한 감정 상태 분석, 요약 및 개인화 케어 추천 기능
- Spring Boot + MySQL 기반 백엔드 직접 설계 및 구현
- 주간/월간 감정 통계 시각화 및 이전 기간 비교 기능

### MindLog
Flutter 기반 AI 감정 관리 애플리케이션

- PM 역할로 프로젝트 전체 기획 및 일정 관리, 핵심 기능 개발 주도
- OpenAI GPT API를 활용한 감정 분석 및 케어 추천 기능 구현
- Firebase Authentication / Firestore 기반 데이터 구조 설계 및 구현
- 감정 통계 시각화 (주간/월간 레이더 차트) 기능 개발

### TodoFriends 🚧 개발중
Android(Kotlin) 기반 AI 소셜 일정 관리 애플리케이션

- OpenAI API를 활용한 개인 및 팀 일정 추천 기능 구현
- Spring Boot + MySQL 기반 REST API 서버 연동
- Jetpack Compose 기반 UI 구현 및 Figma 화면 설계

### GoRoute
React Native 기반 외국인 대상 관광 길찾기 애플리케이션 — 졸업작품

- 독일 교환학생 경험을 바탕으로 외국인의 국내 여행 불편함을 해소하기 위해 기획
- Naver Map API 기반 지도 및 위치 기능 구현
- 관광지 정보 제공 및 다국어 사용자 인터페이스 개발

### EcoBite
Android 기반 냉장고 식재료 관리 애플리케이션

- Google ML Kit OCR 기반 영수증 인식 기능 구현
- 바코드 스캔을 통한 식재료 자동 등록 기능

### Popple 
Android 기반 팝업스토어 정보 제공 애플리케이션 -캡스톤디자인 경진대회 우수상

- Firebase Realtime Database 기반 실시간 채팅 기능 구현
- XML 기반 UI 레이아웃 및 스타일 구현
